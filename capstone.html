Project Summary

Real-time firmware and display integration.

This senior design project focused on capturing ECG signal data, processing samples continuously, and visualizing waveform output through an SPI-based LCD and resistive touchscreen interface.

Core Technologies

  • MSPM0G3507 LaunchPad
  • Embedded C
  • Code Composer Studio
  • ADC sampling
  • SPI LCD
  • Resistive touchscreen

Contributions

What I built

Signal Acquisition

Developed embedded C firmware for real-time signal acquisition and processing using ADC sampling pipelines.

Continuous Capture

Implemented circular buffering to support ongoing waveform capture without interrupting the display workflow.

User Interface

Interfaced the LCD and touchscreen so users could view signal data and select operating modes directly on the device.

Validation

Debugged analog and digital behavior with simulated and live input signals to verify stable system performance.
